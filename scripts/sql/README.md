# SQL Manual e Legado

Esta pasta guarda SQL de apoio que não faz parte da sequência canônica de migrations.

- Use `scripts/migrations/` para mudanças versionadas e ordenadas de schema.
- Use esta pasta para consultas manuais, bootstrap antigo ou referência operacional.
- Antes de promover qualquer arquivo daqui para migration, atribua numeração e valide no Supabase.
