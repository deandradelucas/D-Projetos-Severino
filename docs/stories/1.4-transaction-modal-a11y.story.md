- [x] `role="dialog"`, `aria-modal="true"`, `aria-labelledby` no painel do modal.
- [x] Escape fecha o modal (exceto durante `saving`).

- [x] `document.body` com `overflow: hidden` + classe `horizon-modal-open` enquanto aberto (esconde FAB/bottom nav conforme CSS existente).
- [x] Focus trap com Tab / Shift+Tab dentro do modal.
- [x] Foco inicial no primeiro controlo focável ao abrir.
- [x] Hook reutilizável `src/hooks/useModalA11y.js` (focus trap + lock).
- [ ] Smoke manual: abrir Nova transação → Tab cicla dentro do modal → Escape fecha → FAB inferior some com modal aberto.
