# Story 7.5 — API Route + Web UI de Import

## Status
Done

## Descrição
Criar rota `POST /api/import/planilha` que aceita upload multipart de arquivo Excel ou PDF, e um componente React de upload na página de Transações (ou em página dedicada) com preview do resultado e confirmação antes de salvar. Esta story entrega a experiência de import via interface web — complementar ao WhatsApp (Story 7.4).

## Critérios de Aceite

### Backend — Rota de Import
- [x] Nova rota em `server/routes/register-import.mjs` (novo arquivo)
- [x] `POST /api/import/planilha` — multipart/form-data, campo `arquivo`
- [x] Autenticação via `parseUsuarioEscopoApi` (mesmo padrão das outras rotas)
- [x] Limite de tamanho: rejeita > 20MB com 413
- [x] Validação de mimeType (xlsx, xls, csv, pdf, ofx, qfx) — rejeita outros com 400
- [x] Aceita também por extensão do filename quando mimeType for `application/octet-stream`
- [x] Rate limit: 5 imports por hora por usuário (`import-planilha:{actorId}`)
- [x] Rota registrada em `server/routes/register-all.mjs`

### Backend — Endpoint de Preview (opcional, fase 2)
- [ ] `POST /api/import/preview` — mesmo upload, retorna as primeiras 10 transações parsed SEM salvar
- [ ] Útil para o usuário conferir antes de confirmar
- [ ] **Opcional para esta story** — pode ser implementado junto com UI ou deixado para sprint futura


### Frontend — Componente de Upload

### Frontend — Integração na Página de Transações

### Tratamento de Erros
- [x] 413 (arquivo muito grande): "Arquivo muito grande. Máximo: Excel 10MB / PDF 20MB."

- [x] 429 (rate limit): "Você já importou várias planilhas hoje. Tente novamente amanhã."
- [x] 5xx: "Erro ao processar o arquivo. Tente novamente."

## Notas Técnicas

**Hono multipart:**
```js
const body = await c.req.parseBody()
const file = body['arquivo']  // File object
const buffer = Buffer.from(await file.arrayBuffer())
const mimeType = file.type
```

**Frontend fetch multipart:**
```js
const formData = new FormData()
formData.append('arquivo', file)
const res = await fetch('/api/import/planilha', { method: 'POST', body: formData, headers: { 'x-user-id': userId } })
```

**Hono já suporta multipart nativo** — não precisa de middleware extra
