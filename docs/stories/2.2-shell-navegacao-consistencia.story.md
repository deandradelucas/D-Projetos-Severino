# Story 2.2: Shell & Navegação — Consistência Total

## Status
Ready

## Descrição
Padronizar o shell de navegação (Sidebar desktop, MobileBottomNav, ShellStickyHeaderScroll e MobileMenuButton) usando os tokens definidos na Story 2.1. O shell é o elemento mais visto do app — deve transmitir sofisticação financeira premium em qualquer tamanho de tela.

## Critérios de Aceite
- [ ] `Sidebar.jsx` usa exclusivamente tokens do design system — sem hex/px hardcoded
- [ ] `MobileBottomNav.jsx` com ícones vetoriais nítidos, label legível e item ativo com destaque gold animado (transition 150ms)
- [ ] Item ativo no shell (Sidebar + MobileBottomNav) usa `--color-accent-gold` com indicador visual claro (borda esquerda no desktop, dot/underline no mobile)
- [ ] Hover states em todos os itens de nav: `opacity` + `color` transition em `--duration-fast` (150ms)
- [ ] MobileMenuButton com animação hamburger→X suave (300ms)
- [ ] Shell funciona identicamente em 375px, 768px e 1280px+
- [ ] Tema claro e escuro sem flicker ao navegar
- [ ] `npm run lint` + `npm run build` passam
- [ ] Contrato SHELL HUB em `dashboard.css` (bloco marcado) preservado
