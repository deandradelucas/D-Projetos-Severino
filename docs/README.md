# Docs

Mapa rápido da documentação do Horizonte Financeiro.

- `code-organization.md`: convenções de organização do código.
- `theme-guidelines.md`: contrato visual e temas do dashboard.
- `agenda-whatsapp.md`: operação da agenda integrada ao WhatsApp.
- `stories/`: histórias e auditorias de evolução do projeto.
- `n8n-*.json`: exports de workflows de automação.

Ao adicionar novos documentos operacionais, prefira nomes claros por domínio e referencie scripts ou rotas afetadas.
